# Physics2D-TestBed-Qt

Simple TestBed of 2D Physics Engine For Tutoring.

# Build

```
cmake CMakeLists.txt
```

# Requirement

- C++ 17
- Qt

# Features

- Basic GUI
- Basic Debug Drawing
  - Rigid Body
  - AABB
  - DBVH
  - Joint
- Basic 2D Camera
  - Zooming
  - Smooth Transition
  - Tracing Specified Body
